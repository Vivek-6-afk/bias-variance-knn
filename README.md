# Bias–Variance Trade-off using K-Nearest Neighbour

This repository contains my machine learning tutorial which explains the bias–variance trade-off using the K-Nearest Neighbour (KNN) algorithm.

## Tutorial Objective

The main objective of this tutorial is to understand how the choice of model complexity affects prediction performance.  
In this work, different values of K are analysed to observe how they influence overfitting, underfitting and generalisation ability.

## Contents

This repository includes:

• Tutorial report (PDF)  
• Jupyter notebook with complete experimental code  
• Figures used in the tutorial  
• README and licence file  

## Dataset

A standard classification dataset is used for the experiment.  
The dataset is divided into training and validation sets to evaluate how well the model performs on unseen data.

## How to Run

1. Open the notebook file in Jupyter.
2. Install required libraries if needed:

pip install numpy matplotlib scikit-learn

3. Run all cells step-by-step to reproduce the results and figures.

## Learning Outcome

This tutorial helps learners understand:

• Bias and variance concepts  
• Effect of parameter tuning in machine learning  
• Model generalisation behaviour  
• Practical experimentation using KNN
